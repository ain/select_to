# SelectTo

SelectTo gem is Rails helper that creates `<select>` control for navigation. Optionally for AJAX request.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'select_to'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install select_to
